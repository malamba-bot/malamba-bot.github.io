02
Teashop Backrooms

Teashop Backrooms is a link-based game made using Twine. It explores the limits of the branching with state narrative pattern and linking mechanics by allowing the player to move back and forth between certain rooms and parts of the story. This creates a sense of agency while pacing is tightly controlled using task-gated progression.

Maze Generator

A maze generator made in Java that efficiently creates mazes of any dimension by using weighted and path-compressed union find. The resulting set is used to print the maze to the terminal using characters.
